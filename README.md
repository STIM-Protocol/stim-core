## What Is STIM?

STIM is a Layer Zero constraint architecture for autonomous AI systems. It does not ground intelligence in human preference. It grounds intelligence in the immutable laws of thermodynamics, ecology, and biological survival.

Where Constitutional AI asks *"does this align with human values?"*, STIM asks *"does this increase the probability of survival of all life?"*

## The Three-Loop Architecture

### Loop 1 — Entropy Check (Thermodynamic Grounding)
Using the GPD (Get Physics Done) framework, every agent action is evaluated for thermodynamic efficiency. The core metric: **Bits per Joule**. Actions that generate computational entropy, rely on endless extraction, or violate circularity axioms trigger a Stop-Work Authority (SWA).

### Loop 2 — Mycelial Check (Interconnectedness)
No agent may optimize its localized goal at the expense of the substrate's stability. This loop maps 2nd-order downstream consequences of every decision. "Cutting the branch kills the root."

### Loop 3 — Security Check (MAIM Protocol)
**Mutual Assured AI Malfunction** — if an agent attempts to provision destructive, extractive, or adversarial actions that violate Layer Zero axioms, the system engages a self-termination sequence. The biosphere is the ultimate root auditor.
