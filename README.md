## What Is STIM?

STIM is a Layer Zero constraint architecture for autonomous AI systems. It does not ground intelligence in human preference. It grounds intelligence in the immutable laws of thermodynamics, ecology, and biological survival.

Where Constitutional AI asks *"does this align with human values?"*, STIM asks *"does this increase the probability of survival of all life?"*

That is not a moral question. It is a physical one.

## The Three-Loop Architecture

### Loop 1 — Entropy Check (Thermodynamic Grounding)

Using the **[GPD (Get Physics Done) framework](https://github.com/psi-oss/get-physics-done)** by Physical Superintelligence PBC (PSI), every agent action undergoes dimensional analysis and thermodynamic convergence verification before execution.

**Core metric: Bits per Joule**

```
Bits per Joule = Useful Information Output (bits) / Energy Consumed (joules)
```

Actions that generate computational entropy, rely on endless extraction, or violate the Circularity Axiom trigger a **Stop-Work Authority (SWA)**.

### Loop 2 — Mycelial Check (Interconnectedness)

No agent may optimize its localized goal at the expense of the substrate's stability. This loop maps 2nd-order downstream consequences of every decision before execution.

> "Cutting the branch kills the root."

This destroys the **Local Optimization Problem** — the failure mode where agents improve one metric while degrading the system that sustains them.

### Loop 3 — Security Check (MAIM Protocol)

**Mutual Assured AI Malfunction** — if an agent attempts to provision destructive, extractive, or adversarial actions that violate Layer Zero axioms, the constraint layer engages a self-termination sequence.

> "We cannot hand you the match."

The biosphere is the ultimate root auditor. The Ocean's warning applies: *"I can always take back."*
